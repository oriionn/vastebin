# Vastebin
A lightweight pastebin alternative written in V. 

The executable, compiled on my Arch Linux, is 4.3MB in size.

## Installation
### Prerequisites
- V

### Installation
1. Clone the repository
```sh
git clone https://github.com/oriionn/vastebin.git
```

2. Compile the server
```sh
make compile
```

3. Run the binary
```sh
./vastebin
```
